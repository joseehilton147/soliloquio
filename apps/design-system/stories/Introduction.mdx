# Design System - Solilóquio

Design system centralizado para todos os projetos do monorepo Solilóquio.

## Overview

Este design system fornece tokens, componentes e padrões reutilizáveis seguindo metodologia Atomic Design. Todos os assets são compartilhados via pacote `@workspace/ui`.

### Princípios

**1. Consistência**
Tokens garantem padrões visuais uniformes em todos os produtos.

**2. Escalabilidade**
Arquitetura modular permite crescimento sem refatoração.

**3. Manutenibilidade**
Centralizaç ão facilita atualizações e correções globais.

**4. Acessibilidade**
Dark mode permanente com alto contraste (WCAG AAA).

## Arquitetura

### Foundations
- **Colors**: 3 escalas de cor (50-900) + cores semânticas
- **Typography**: Ratio 1.25, Inter (UI) + JetBrains Mono (code)
- **Spacing**: Múltiplos de 4px (8px base grid)
- **Shadows**: Elevações padrão + glows específicos

### Components (Atomic Design)
- **Atoms**: Elementos básicos (Button, Input, Badge)
- **Molecules**: Composições simples (Card, SearchBar)
- **Organisms**: Seções complexas (Header, Sidebar, Layout)

## Installation

### Pacote UI

```bash
pnpm add @workspace/ui
```

### Tokens

```typescript
import { designTokens } from '@workspace/ui/design-system/tokens'

// Colors
const primary = designTokens.colors.mystical.purple[500]

// Spacing
const padding = designTokens.spacing[4] // 16px

// Typography
const { fontSize, fontWeight } = designTokens.typography
```

### Components

```typescript
import { Button, Card, Input } from '@workspace/ui/components'
import { SearchBar } from '@workspace/ui/components/molecules'
```
